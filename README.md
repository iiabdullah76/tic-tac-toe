Classic Tic-Tac-Toe (HTML/CSS/JS) with win detection and replay. Clean, beginner-friendly code.

## Demo
https://iiabdullah76.github.io/tic-tac-toe/

## Run locally
1. Download ZIP or  
   `git clone https://github.com/iiabdullah76/tic-tac-toe.git`
2. Open `index.html` in your browser.
